### Installation
```bash
$ [sudo] pip install mktouch
```

```bash
$ [sudo] npm i -g mktouch
```

#### Examples
```python
>>> import mktouch
>>> mktouch.mktouch("folder/file")
>>> mktouch.mktouch(["folder/file1","folder/file2"])
```

```bash
$ mktouch /tmp/not-existing-dir/file
$ ls /tmp/not-existing-dir/file
/tmp/not-existing-dir/file
```
